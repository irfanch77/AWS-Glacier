== Description ==

This is a simple plugin to backup and store your WP Site and Database to Amazon Web Services Glacier which is meant for long term storage. Must have an active AWS Glacier Account.

== Installation ==


1. Upload into to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

=== Requirement ===

PHP 5.3.1 and above
wordpress 3.0.1
Mysql
